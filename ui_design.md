# 问卷系统页面布局与交互设计 (UI Design)

基于第一阶段实验要求与微拟态前端模板，系统主要包含以下四个核心页面。为了完美支持“题目跳转逻辑”并维持清晰的状态管理，问卷填写端采用**单题翻页（卡片式）**交互。

## 1. 仪表盘页面 (Dashboard)

**定位：用户的控制台，管理所有本人创建的问卷。**
**视觉风格：** 清爽卡片风，突出操作按钮。

```text
============== 【顶部导航栏】 ==============
AI 问卷系统               欢迎, wcx | [退出登录]
========================================

【 ➕ 创建新问卷 】 (大号主操作按钮)

我的问卷列表 (总计: 2)
┌──────────────────────────────────────────────┐
│ 📝 问卷标题：大学生学习习惯调查
│ 状态：🟢 收集中  |  答卷数量：128份
│ ──────────────────────────────────────────── │
│ [编辑问卷]   [📋分享链接]   [📊查看统计]   [关闭问卷]
└──────────────────────────────────────────────┘
┌──────────────────────────────────────────────┐
│ 📝 问卷标题：2026产品满意度反馈
│ 状态：🟡 暂未发布 |  答卷数量：0份
│ ──────────────────────────────────────────── │
│ [编辑问卷]   [发布问卷]   [📊查看统计]   [删除问卷]
└──────────────────────────────────────────────┘
```

## 2. 问卷编辑页面 (Survey Editor)

**定位：创建问卷、添加题目、设置跳转逻辑的后台页面。**
**视觉风格：** 表单区块划分，题目按顺序纵向排列，选项支持下拉配置跳题逻辑。

```text
============== 【顶部导航栏】 ==============
[ < 返回仪表盘 ]       [保存草稿] [ ✅ 发布问卷]
========================================

[ 基础设置卡片 ]
- 问卷标题： [ 输入框：_____________________ ]
- 问卷说明： [ 文本域：_____________________ ]
- 高级设置： ☑️ 允许匿名填写    截止时间: [ 日期选择 ]

[ 题目管理区 ]
┌──────────────────────────────────────────────┐
│ Q1. 你的性别 (单选题)                    [删除] │
│ ☑️ 必答题
│ 选项A：男  ──> [跳转至：下一题 ▼] (下拉配置选择)  │
│ 选项B：女  ──> [跳转至：Q3     ▼] (设置了跳题)   │
│ [ ➕ 添加选项 ]
└──────────────────────────────────────────────┘
┌──────────────────────────────────────────────┐
│ Q2. 你的年龄 (填空题 - 数字)             [删除] │
│ ☑️ 必须为整数   最小值: [ 0 ]   最大值: [ 120 ]       │
└──────────────────────────────────────────────┘

(悬浮操作条)
[+ 添加单选题] [+ 添加多选题] [+ 添加文字填空] [+ 添加数字填空]
```

## 3. 问卷填写页面 (Survey Fill)

**定位：呈现给被调查者的前台答题端。**
**交互模式：** **单题翻页卡片式**（完美解决长列表跳题带来的DOM隐藏折叠与状态回退问题）。
**视觉风格：** 采用 `pretest-board-preview.html` 的微拟态玻璃质感样式，背景为渐变色+动态气泡。

```text
(全屏背景：N2W渐变色、缓慢漂浮的毛玻璃气泡图案)

        ╭──────────────────────────────────────╮
        │  ==== 进度条：1/5 题 ==== (随进度填充)   │
        │                                      │
        │  Q1. 你的性别是什么？（单选）                   │
        │                                      │
        │  [   A. 男                           ]       │
        │  [✅ B. 女   (选中时带高亮和内阴影反馈) ]       │
        │                                      │
        │  (如果有错误校验，此处显示：)                     │
        │  [ ⛔ 选项必答，请选择一个答案！     ]       │
        │                                      │
        │  [ 上一题 ]                    [ 下一题 ]       │
        ╰──────────────────────────────────────╯

* 交互说明：
1. 点击“下一题”时，前端根据当前题目的所选项及跳转逻辑（如选B跳Q3），直接重新渲染卡片内容为目标题目，实现状态机式的稳健跳题。
```
